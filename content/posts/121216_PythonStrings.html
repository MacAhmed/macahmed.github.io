Google - Python Strings

12 Dec 2016

Strings are a sequence of characters and are found in almost every language. They are mostly used to represent textual information. Python allows you to create a string literal using " and '. However, the delimiter must be the same set. So, for example : "Mac" and 'Mac' are both viable strings while 'Mac" or "Mac' are not.

The strings acts similar to array where each character can be accessed using an index. Indexing of a string starts from 0 to length(string). The general form to access a character of a string is <string>[<expr>]. For Example : 

greet = "Mac"
greet[0] = 'M'
greet[2-1] = "a"

A substring can be accessed using the slicing operation. so, greet[0:3] = 'Hel' therefore, generally, <string>[<start>:<end>]. + can be used to concatenate strings while * can be used to multiply the string out.

For Example, if s = "string" then s = s + s = "stringstring", and s = 3*s = "stringstringstring".

There are some common methods in the Python libraries for string manipulation.

Method Description
s.lower() , s.upper() returns the lowercase or uppercase version of the string.
s.strip() returns a string with whitespace removed from start and end
s.isalpha() / s.isdigit() / s.isspace() tests if all the string chars in the various character classes
s.startswith('other'), s.endswith('other') tests if the string starts or ends with a certain string
s.find('other') searches for the given other string ( not a regex ), and returns the index where it starts
s.replace('old', 'new') returns a string where all occurneces of 'old' have been replaced with 'new'
s.split('delim') returns substrings separated with the given delimiter ( not a regex ). The s.split without arguments splits on whitespace.
s.join(list) opposite of split, and joins the list into a string using a delimiter.

More methods can be found on python.org string methods
