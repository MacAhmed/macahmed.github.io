Python's lists are ordered sequences of items. List literals are often written in []. So, For Example : 

colors = ['red', 'blue', 'green']
print colors[0]          ## red
print colors[2]          ## green
print len(colors)        ## 3


Assignment or = on lists does not make a copy of the list but rather assigns the same address to that variable. Empty list is defined using [] and adding or the + operator concatenates the two lists. For Example : [1 , 2] + [3 , 4] results in [1 , 2 , 3 , 4].

The for construct -- for var in list -- is a easy way to look at each element in a list. for and in can both be used with strings as well. The range() function can be used to implement foreach loops.

Python's while loop is fairly standard and the break and continue statements work the same as in Java.
For example : Here is a while loop that assesses every second element in a list. 

i = 0
while i < len(a)
    print a[i]
    i = i + 2

Here are some common methods :

Method Description
list.append(elem) Appends the element to the end of the list.
list.insert(index, elem) inserts the element at the specified index of the list
list.extend(list2) add all elements of list2 to the end of list
list.index(elem) returns the index of that element in the list.
list.remove(elem) removes the element from the list
list.sort() sorts the list in place
list.reverse() reverses all items in the list
list.pop(index) removes and returns the element at the specified index
